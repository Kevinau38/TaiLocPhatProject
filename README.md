# Tài Lộc Phát Showroom

Website hiện đại cho doanh nghiệp vật liệu xây dựng tại TP. Hồ Chí Minh.

## Khởi chạy
```bash
bun install
bun run dev
```

## Triển khai
```bash
bun run build
bun run deploy
```
